# Installation

To install Toybox, run this command in your terminal:
```
composer create-project toybox/theme toybox --stability=dev --remove-vcs
```

You may also want to install the Toybox plugin. You can do so by [downloading the plugin](https://toybox.maxweb.cloud/toybox/latest) and unzipping it in the `wp-content/plugins` directory, or [uploading the plugin to your WordPress installation via the WordPress admin panel](/wp-admin/plugin-install.php).

## Configuring the plugin

Once you've installed the plugin, you'll need to activate it in the WordPress admin panel. Navigate to the Plugins page and click the Activate button next to the Toybox plugin.

Once you've activated the plugin, you'll need to configure Toybox. Navigate to the Settings page and click the Toybox link in the left sidebar. You'll be taken to the Toybox settings page. From here, you can login to the Toybox API.

## Registering an environment

After you have logged into the API, you can register an environment. To do so, go into the Environments tab and click the Register Environment button. A form will appear where you can enter the environment name. Please ensure the environment name is unique and valid to the actual environment it is on.
